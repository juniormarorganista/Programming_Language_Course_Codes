# Mathematica

Esta trilha organiza o estudo de Mathematica/Wolfram Language com foco em:
- sintaxe básica;
- álgebra simbólica;
- cálculo simbólico;
- gráficos;
- pequenos projetos;
- reaproveitamento de material de pesquisa.

## Estrutura

- `algebra/`
- `calculus/`
- `plots/`
- `projects/`
- `research/`

## Como executar

```bash
wolframscript -file caminho/do/arquivo.wls
```
