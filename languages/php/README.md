# PHP

## Objetivo

Aprender PHP do zero até uma base sólida para web e scripts simples.

## Estrutura do módulo

```text
languages/php/
├── README.md
├── basics/
├── functions/
├── arrays/
├── forms/
├── oop/
└── mini_projects/
```

## Ordem sugerida

1. sintaxe básica

3. condicionais
4. laços
5. funções
6. arrays
7. formulários
8. POO
9. mini projetos

## Como executar no Windows

### Script CLI
```powershell
php .\languages\php\basics\01_hello.php
```

### Servidor local simples
```powershell
php -S localhost:8000 -t .\languages\php\forms
```
