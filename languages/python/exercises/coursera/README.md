# Acervo Coursera

Este diretório preserva os códigos originais do curso já existentes no repositório.

## Como usar este material

- trate como acervo histórico;
- não apague os arquivos antigos;
- quando melhorar um exercício, prefira criar uma nova versão documentada ao lado ou em `guided/`;
- use commits específicos para revisões.

## Estratégia prática

1. execute um arquivo antigo;
2. entenda o que ele resolve;
3. reescreva com nomes melhores e comentários mais úteis;
4. compare a versão antiga com a nova.
