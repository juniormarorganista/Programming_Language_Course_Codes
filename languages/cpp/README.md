# C++

A trilha de C++ parte da base procedural e avança para recursos mais modernos da linguagem.

## Estrutura

- `basics/`: fundamentos com STL, classes, referências e exceções.
- `projects/`: mini projetos de terminal.
- `projects/calculator/`: calculadora original reorganizada.
