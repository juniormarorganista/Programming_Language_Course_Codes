# Trilha de estudo: C, C++ e Fortran

Esta etapa foi pensada para consolidar três frentes importantes:

- **C** para base de memória, ponteiros, arrays e raciocínio de baixo nível.
- **C++** para transição da base procedural para STL, classes e organização moderna.
- **Fortran** para computação científica, arrays numéricos e I/O em contexto mais próximo de códigos técnicos.

## Estratégia de uso

A lógica é simples:

1. Estude os arquivos em `basics/` na ordem numérica.
2. Compile e rode cada arquivo separadamente.
3. Leia o cabeçalho de cada exemplo.
4. Faça pequenas alterações por conta própria.
5. Só depois avance para `projects/`.

## Meta desta fase

No fim desta trilha, você deve ser capaz de:

- compilar pequenos programas nativos no Windows;
- diferenciar ponteiros, referências e arrays em cada linguagem;
- escrever programas de terminal com entrada, saída e arquivos;
- começar a organizar mini projetos sem virar refém de IDE.
