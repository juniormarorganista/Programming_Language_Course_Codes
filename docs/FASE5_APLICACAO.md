# Como aplicar a fase 5

1. Copie este pacote para a raiz do repositório.
2. Rode `scripts/validate-phase5.ps1`.
3. Se tiver Octave, R e/ou Wolfram instalados, rode `scripts/run-phase5-examples.ps1`.
4. Revise o `git status`.
5. Faça o commit da fase.

## Resultado esperado

Ao final, o repositório terá:
- trilha de MATLAB/Octave;
- trilha de R;
- trilha de Mathematica;
- documentação transversal adicional;
- scripts de validação e execução para Windows.
