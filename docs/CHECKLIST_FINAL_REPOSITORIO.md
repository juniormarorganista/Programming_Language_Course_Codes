# Checklist Final do Repositório

## Estrutura
- [ ] `docs/` está consistente
- [ ] `languages/` está consistente
- [ ] cada linguagem tem README
- [ ] scripts PowerShell estão presentes
- [ ] `.github/` está útil

## Qualidade
- [ ] `.gitignore` não sabota documentação
- [ ] não há binários indevidos rastreados
- [ ] exemplos novos têm cabeçalho padrão
- [ ] nomes de arquivos seguem convenção numérica quando aplicável

## GitHub
- [ ] existe branch da fase
- [ ] existe PR da fase
- [ ] existem issues para módulos futuros
- [ ] existe release do marco atual

## Aprendizado
- [ ] pelo menos uma linguagem base foi fortalecida
- [ ] pelo menos uma linguagem nova foi iniciada
- [ ] houve execução real de exemplos
- [ ] documentação foi atualizada junto com o código
