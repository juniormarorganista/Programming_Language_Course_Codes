## Visão geral

Este repositório reúne códigos de estudo, exercícios, miniaplicações e exemplos técnicos em diferentes linguagens de programação. A ideia não é só mostrar sintaxe solta: é mostrar **organização**, **variedade**, **base conceitual** e **capacidade de transformar estudo em material reaproveitável**.

O portfólio foi reorganizado para funcionar como:

- trilha de estudo por linguagem;
- vitrine técnica para GitHub;
- base para revisão de conceitos;
- conjunto de exemplos fáceis de executar e expandir.

## O que este repositório demonstra

- fundamentos sólidos de programação;
- contato com linguagens de propósito geral e científico;
- organização por diretórios, temas e níveis de dificuldade;
- exemplos executáveis em Python, C, C++, Fortran, R, SQL, MATLAB/Octave e Mathematica;
- material útil tanto para estudo quanto para entrevistas e portfólio técnico.

## Como começar rápido

### Python
```bash
cd Python
pip3 install -r requirements.txt
python3 Foundations/hello_world.py
python3 OOP/bank_account.py
```

### SQL com SQLite
```bash
cd SQL
sqlite3 portfolio.db < 01_schema.sql
sqlite3 portfolio.db < 02_insert_data.sql
sqlite3 -header -column portfolio.db < 10_reports.sql
```

### C
```bash
cd C
make help
make calculator
./calculator
```

### C++
```bash
cd CPP
make help
make calculator
./calculator_cpp
```

### Fortran
```bash
cd Fortran
make help
make bisection
./bisection
```
