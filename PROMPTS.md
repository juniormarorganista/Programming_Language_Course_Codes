# PROMPTS.md — Programming_Language_Course_Codes

## Uso econômico no Plus
- reorganize uma linguagem por vez
- priorize estrutura e README antes de criar muitos exemplos novos
- preserve o que já existe e melhore progressivamente

## Prompt 1 — auditoria estrutural
Audite este repositório sem editar arquivos.

Quero:
1. a estrutura atual por linguagem
2. temas que já existem
3. arquivos duplicados ou mal posicionados
4. lacunas de README e instruções de execução
5. exemplos que valem ser preservados
6. uma estrutura-alvo mais pedagógica
7. um plano em 5 passos pequenos

## Prompt 2 — reorganização por linguagem
Implemente a reorganização apenas para uma linguagem escolhida por você entre as mais maduras do repositório.
Não mexa nas demais ainda.
Crie estrutura pedagógica progressiva e atualize o README correspondente.

## Prompt 3 — README por linguagem
Escolha uma linguagem e proponha:
- objetivos do módulo
- ordem dos tópicos
- instruções de execução
- próximos exemplos a criar

Não edite arquivos ainda.

## Prompt 4 — limpeza mínima
Proponha uma limpeza mínima para:
1. nomes de arquivos confusos
2. exemplos duplicados
3. falta de instruções de execução
4. arquivos órfãos

Não edite arquivos ainda.

## Prompt 5 — primeira implementação
Implemente apenas a organização inicial da linguagem Python,
preservando o máximo possível do conteúdo existente
e adicionando README e instruções de execução.
