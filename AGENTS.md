# AGENTS.md — Programming_Language_Course_Codes

## Objetivo
Organizar códigos de estudo e prática por linguagem de programação, com estrutura pedagógica clara e reuso simples.

## Regras de organização
- Organize por linguagem e depois por tema.
- Cada exemplo deve ensinar um conceito principal.
- Evite arquivos gigantes com muitos assuntos misturados.
- Cada pasta principal deve ter README com objetivo, execução e pré-requisitos.
- Preserve exemplos pequenos, claros e executáveis.
- Mantenha textos e READMEs em português.

## Estrutura preferida
- uma pasta por linguagem
- subpastas por módulo ou tema
- README por linguagem
- exemplos mínimos e progressivos
- exercícios quando fizer sentido

## Validação
- todo exemplo deve indicar como executar
- scripts devem rodar sem edição manual quando possível
- evitar dependências desnecessárias
- preservar exemplos existentes quando forem úteis

## Saída esperada
1. o que foi alterado
2. quais arquivos mudaram
3. como executar
4. riscos remanescentes
5. pendências
