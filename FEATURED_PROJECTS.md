# Projetos em destaque

Este arquivo aponta os diretórios que valem abrir primeiro quando a ideia é entender rapidamente o que o portfólio entrega.

## 1. Python/Apps
**Por que abrir:** mostra Python com interface, visualização e reaproveitamento de código.

**Arquivos principais:**
- `app_streamlit.py`
- `app_tkinter.py`
- `graficos_funcoes.py`

**Destaque técnico:** separação entre lógica de plotagem e interface.

## 2. Python/Scientific_Computing
**Por que abrir:** conecta programação com arrays, matrizes, gráficos e raciocínio quantitativo.

**Arquivos principais:**
- `numpy_arrays.py`
- `matrix_operations.py`
- `simple_plot.py`

## 3. SQL
**Por que abrir:** trilha curta, clara e ótima para mostrar domínio básico-intermediário de consultas.

**Arquivos principais:**
- `01_schema.sql`
- `06_joins.sql`
- `10_reports.sql`

## 4. C/Calculator
**Por que abrir:** pequeno projeto modular em C, com organização em múltiplos arquivos.

**Arquivos principais:**
- `main.c`
- `calculator.c`
- `calculator.h`

## 5. Fortran/Numerical_Methods
**Por que abrir:** reforça o eixo de computação científica do repositório.

**Arquivo principal:**
- `bisection_method.f90`
